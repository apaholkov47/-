# Mobile Application

## Назначение
Мобильное приложение для заказа и отслеживания доставки еды.

## Планируемые компоненты
- Авторизация пользователя
- Каталог ресторанов
- Корзина заказов
- Оплата
- История заказов

## Взаимодействие
Приложение взаимодействует с ServerApi через REST API.
Получает данные о ресторанах, заказах и пользователях.
